# Тестування працездатності системи

# Засоби тестування

Тесутвання API виконувалося за допомогою програми-сервісу Postman яка дозволяє робити запроси типу GET, POST та має зручний інтерфейс користувача.

# Ремарка

Тести, які можна побачити тут - лише частина для демонстрації того, що API працює Для перевірки інших запросів, потрібно розгорнути версію локально та тестувати за допомогою сервісу для тестування API. Під час виконання роботи для тестування був застосований сервіс Postman
